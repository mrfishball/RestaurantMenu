A simple restaurant menu web app that utilizes SQLite 3 and Python.

File List:

-CRUDCreate.py --example of a create operation
-CRUDUpdate.py --example of an update operation
-database_setup.py --setup and class file for the restaurant menu database
-lotsofmenu.py to --populate menus for the database
-restaurantmenu.db --database file
-webserver.py --a custom webserver to host the the database

How to(in vagrant environment):

-run python webserver.py
-type localhost:8080/restaurants to access the front page
-fully functional web app that allows CRUD operations on a database!
